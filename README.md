# Matic documentation

It contains all documentations for Matic Network and Matic products.

### Usage

**Installation**

```bash
# Create and activate virtualenv
$ virtualenv venv
$ source venv/bin/activate
$ pip install -r requirements.txt
```

**Development**

```bash
# serve on localhost
$ mkdocs serve
```

**Production**

```bash
# build docs
$ mkdocs build --clean
```

**Deploy on Matic server**

```bash
$ bash deploy.sh
```
